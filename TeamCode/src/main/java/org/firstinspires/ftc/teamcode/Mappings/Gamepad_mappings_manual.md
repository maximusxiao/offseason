### Gamepad 2 - Gameplay
* Left Joystick
<pre>
  Movement of Extendo. Up = outward, down = inward; Sideways movement of joystick not configured to function.
</pre>
* Right Joystick
<pre>
  Movement of lift. Up = upward, down = downward; Sideways movement of joystick not configured to functon
</pre>
* Right Trigger
<pre>
  Spinning intake mechanism
</pre>
* Left Trigger
<pre>
  Rejection of block in intake
</pre>
* Left Bumper
<pre>
  Used for rotation of arm controlling intake pod; alternatively up and down(single button)
</pre>
* Right Bumper
<pre>
  Used for adjusting pitch of intake pod; alternatively reject and intake positions(single button)
</pre>
* Button A
<pre>
  Used to set state of lift arm
</pre>
* Button B
<pre>
  Used to set state of lift arm
</pre>
* ~~Button X~~
<pre>
  Used to set state of lift arm
</pre>
* Button Y
<pre>
  Used to set state of lift arm
</pre>
* D-pad
<pre>
  Up = PTO mode, down = stilts, left = claw, right = wrist
</pre><br />
